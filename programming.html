مقدمه‌ای بر برنامه‌نویسی

برنامه‌نویسی یعنی بیان یک راه‌حل برای کامپیوتر، با زبانی که قابل اجرا باشد. در همه زبان‌ها مفاهیم مشترکی مثل متغیر، نوع داده، ساختارهای کنترلی (شرط و حلقه) و توابع وجود دارد.

قدم‌های شروع

  1. انتخاب زبان مناسب (مثل Python یا C++)
  2. یادگیری مفاهیم پایه
  3. حل تمرین‌های کوچک و تدریجی

* این صفحه صرفاً محتوای نمونه برای پروژه دانشجویی است.

مفاهیم پایه برنامه‌نویسی

متغیرها و انواع داده

متغیرها محل ذخیره اطلاعات در حافظه هستند. انواع مختلفی دارند:

  • اعداد صحیح (int)
  • اعداد اعشاری (float)
  • متن (string)
  • منطقی (boolean)

ساختارهای کنترلی

کنترل جریان اجرای برنامه:

  • شرط‌ها (if/else)
  • حلقه‌ها (for/while)
  • سوییچ (switch)

مثال‌های عملی

برنامه ساده "سلام دنیا"

# Python
print("سلام دنیا!")

# JavaScript
console.log("سلام دنیا!");

# C++
#include <iostream>
int main() {
    std::cout << "سلام دنیا!" << std::endl;
    return 0;
}
